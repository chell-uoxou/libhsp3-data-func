# libhsp3_data_func
HSPで複雑なデータやノートパッド命令を扱うためのライブラリです。

## Supported version
Windows版HSP3.4以降に対応しています。Linux版やHSP3Dish版では動作確認しておりません。

## How to use?
HSPがインストールされているフォルダ下の、「common」というフォルダ下に「libhsp3_data_func.as」をコピーしてください。
この状態で、コードを書くときに、先頭に、  
`#include "libhsp3_data_func.as"`  
と記述すると使用できます。

## Note
